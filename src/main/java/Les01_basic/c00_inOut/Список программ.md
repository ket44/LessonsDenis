# Список программ для разработки на Java
## Рассмотрено для Windows

1. Иметь права администратора, но работать безопаснее из учётной записи с ограниченными правами
2. Установить пакет разработки на Java - JDK (Java Development Kit). 
   1. Ставим LTS версию, это JDK 25 (LTS - длительный срок поддержки)
   2. Знать про права использования. Есть варианты, когда использовать программы можно только внутри компании. Хороший вариант OpenJDK 25
   3. При установке включить изменение системной переменной Path 
3. Установить среду разработки. Она сложная, к ней надо привыкать, учиться пользоваться. Рекомендуется OpenIde, допустимы GigaIde, IntelliJ Idea, VS Code с расширением разработки на Java
