# Ключевые концепции для запоминания (Ветвление):
Конструкции ветвления - это реализация ВЫБОРА пути исполнения программы.

Важно: В ветвлении есть код, который может не выполняться!

@before Знать: логические выражения, логические функции.

***************************************
if - базовая условная конструкция

if-else - выбор из двух вариантов

if-else if-else - множественный выбор
***************************************

Вложенные if - условия внутри условий

switch-case - выбор из множества дискретных значений
Тернарный оператор - компактная запись простых условий

Логические операторы:

&& (И) - оба условия истинны

|| (ИЛИ) - хотя бы одно условие истинно

! (НЕ) - инверсия условия
